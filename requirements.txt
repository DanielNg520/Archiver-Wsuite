# Archiver Suite — aggregated runtime dependencies.
#
# The suite is four installable packages (core, archiver, recorder,
# dispatcher; ops has none). The source of truth is each package's
# pyproject.toml — this file is a convenience union for setting up a single
# combined environment (CI, a fresh machine, a venv covering the whole suite).
#
# Constraints below are the UNION of the per-package requirements; where two
# packages pin the same dependency, the tighter/overlapping bound is used.
#
# NOTE: playwright additionally needs a one-time browser download:
#     python -m playwright install chromium
